# デプロイ手順（サイトを立ち上げる人向け）

このドキュメントは「赤ちゃんお祝いアルバムサイト」を、無料のレンタルサーバー
（PythonAnywhere）に公開して、友達に共有URLを送れるようにするための手順。

この`for_gonda`フォルダは、Gondaちゃん用のサイト専用の一式（`config.py`の
`BABY_NAME`はすでに`"Gonda"`に設定済み）。もう1人（Shiaちゃん）用は
別フォルダ`for_shia`にまとめてあるので、そちらは**別のPythonAnywhere
無料アカウント（別のメールアドレス）**で公開すること。**PythonAnywhereの
無料アカウントは1人につきWebアプリを1つしか公開できないため。**

---

## 準備するもの

- `for_gonda`フォルダ一式（`app.py`, `config.py`, `templates/`, `static/`,
  `requirements.txt`など）を1つのzipファイルにまとめたもの
- PythonAnywhereの無料アカウント（<https://www.pythonanywhere.com/> で
  メールアドレスだけで作れる）

---

## ステップ1: PythonAnywhereにアカウントを作る

1. <https://www.pythonanywhere.com/> を開く
2. 「Pricing & signup」→ 無料プラン（Beginner）を選んでアカウントを作る
3. ログインすると、ダッシュボード（自分の作業画面）が出る

---

## ステップ2: zipファイルをアップロードして展開する

1. ダッシュボード上部の「Files」タブを開く
2. zipファイル（例: `for_gonda.zip`）をアップロードする
3. 「Consoles」タブ（または「Files」から「Open Bash console here」）を開き、
   **PythonAnywhereの画面の中に出てくる黒い入力欄（ブラウザ上のターミナル。
   自分のパソコンのコマンドプロンプト/PowerShellではない）** に、以下を入力する

   ```
   unzip for_gonda.zip -d for_gonda
   ```

4. 続けて、必要なライブラリを入れる

   ```
   pip3.10 install --user -r for_gonda/requirements.txt
   ```

   ※これも**PythonAnywhereのブラウザ上のコンソール**で実行する。自分の
   パソコンのターミナルで実行するとエラーになる。

---

## ステップ3: Webアプリを作る

1. 「Web」タブを開き、「Add a new web app」を押す
2. 「Next」→「Manual configuration」（自動設定ではなく手動設定）を選ぶ
3. Pythonのバージョンを選ぶ（3.10系など、pip installで使ったものと合わせる）
4. 作成が終わると、「Code」欄あたりに
   `WSGI configuration file` というリンクが表示される
   （これは`/var/www/ユーザー名_pythonanywhere_com_wsgi.py`という、
   **決まった名前のファイル**で、これ以外の名前のファイルは読み込まれない）

---

## ステップ4: WSGI設定ファイルを書き換える

1. ステップ3で出てきた`WSGI configuration file`のリンクをクリックして、
   編集画面を開く
2. 中身を全部消して、以下の内容に書き換える（`mokuo`の部分は、この
   `for_gonda`用に作った自分のPythonAnywhereのユーザー名に置き換える）

   ```python
   import sys
   path = '/home/mokuo/for_gonda'
   if path not in sys.path:
       sys.path.insert(0, path)

   from app import app as application
   ```

3. 画面右上あたりの「Save」ボタン（緑）で保存する
   （**「Save as...」ではなく「Save」を押すこと**。別名で保存すると、
   PythonAnywhereはそのファイルを読み込んでくれない）
4. もしブラウザの自動翻訳機能が働いていたら、コードが日本語に
   翻訳されて壊れてしまうことがある。その場合はブラウザの翻訳アイコンから
   「原文を表示」「翻訳をやめる」を選んでから、もう一度貼り直す

---

## ステップ5: 公開する

1. 「Web」タブに戻り、緑色の「Reload」ボタン（自分のユーザー名の
   web appの再読み込みボタン）を押す
2. 表示されているURL（例: `https://mokuo.pythonanywhere.com/`）を開いて、
   サイトが表示されれば成功
3. そのURLを友達に共有すれば、写真とメッセージを投稿してもらえる
   （投稿する側の使い方は`かんたんガイド.md`を参照）

---

## うまくいかないとき

- **エラー画面が出る** → 「Web」タブ下部の「Error log」を確認する
- **WSGIファイルの内容が保存されていないように見える** → 保存した
  ファイル名が`/var/www/ユーザー名_pythonanywhere_com_wsgi.py`に
  なっているか、再度「WSGI configuration file」のリンクから確認する
- **コードが日本語に翻訳されて崩れている** → ブラウザの自動翻訳を
  オフにしてから、コードを貼り直して保存し直す
