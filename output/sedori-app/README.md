# せどり仕入れ判断支援アプリ

商品のバーコードや写真から、仕入れたときに「いくら利益が出るか」を
その場で計算できる、スマホ向けWebアプリ（PWA）。

## 使い方（URLを開くだけ）

スマホやパソコンのブラウザで、次のURLを開く。

**https://motonori.pythonanywhere.com**

インストール不要。開いたらすぐ使える。

### できること

1. **バーコードをスキャン** — 商品のJAN/EANバーコードをカメラで読み取る
   （読み取りは端末内で完結。無料）
2. **写真から商品を判別（AI）** — バーコードが無い商品は、写真を撮ると
   AIが商品名の候補を提案する（※判別1回ごとにAPI利用料が数円程度かかる）
3. **利益計算** — 仕入れ価格・想定売価・手数料率・送料を入れると、
   利益額と利益率をその場で計算する
4. **商品の保存・一覧** — 計算した商品を保存して、あとから一覧で見返せる
5. **ECサイトで価格を調べる** — Amazon・メルカリ・Yahoo!・楽天の
   検索結果ページへのリンクを開く（価格の自動取得はしない）

### スマホのホーム画面に追加する（任意）

ブラウザのメニューから「ホーム画面に追加」を選ぶと、アプリのように
アイコンから起動できる。

## 注意事項

- **登録データは全員で共有される**（ログイン機能なし）。URLを知っている人は
  誰でも閲覧・登録・削除ができる。
- 利益計算はあくまで概算。実際の仕入れ・販売の判断は、各ECサイト・店舗の
  最新情報を自分で確認して行うこと。
- 中古品を継続的に営利目的で転売する場合は、**古物営業法上の古物商許可**が
  必要になることがある（アプリ内でも注意喚起を表示している）。

## 表示が古いとき

以前開いたときの画面が保存されていて、更新が反映されないことがある。
その場合は**ページを2回続けて再読み込み**すると最新版になる。
